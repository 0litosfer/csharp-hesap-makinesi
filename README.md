C# Konsol Hesap Makinesi

Bu proje, C# programlama dili kullanılarak geliştirilen konsol tabanlı bir hesap makinesi uygulamasıdır. Kullanıcının seçtiği matematiksel işlemleri gerçekleştirerek sonucu ekrana yazdırır.

Kullanılan Teknolojiler
C#
.NET Console Application
Özellikler

Uygulama aşağıdaki matematiksel işlemleri desteklemektedir:

Toplama işlemi
Çıkarma işlemi
Çarpma işlemi
Bölme işlemi
Sayının karesini alma
Sayının karekökünü alma
Proje İçeriği

Bu projede C# temel programlama yapıları kullanılmıştır:

Kullanıcıdan veri alma
Değişken kullanımı
Switch-case yapısı
Koşul ifadeleri
Matematiksel işlemler
Metot kullanımı
Hata kontrolü
Çalışma Mantığı

Program çalıştırıldığında kullanıcıya işlem seçenekleri sunulur. Kullanıcı yapmak istediği işlemi seçtikten sonra gerekli değerleri girer ve hesaplanan sonuç konsol ekranında gösterilir.

Örnek menü:

1 - Toplama
2 - Çıkarma
3 - Çarpma
4 - Bölme
5 - Kare Alma
6 - Karekök Alma

Seçiminiz:
Kurulum ve Çalıştırma
Repoyu bilgisayarınıza klonlayın:
git clone proje-linki
Projeyi Visual Studio ile açın.
Projeyi çalıştırarak konsol üzerinden kullanmaya başlayabilirsiniz.
